Um pouco sobre SVG

Podemos falar que o SVG usa primitivas geométricas como pontos, linhas, curvas e polígonos para representar a imagem. Chamamos de gráficos vetoriais. A imagem em SVG não perde a sua qualidade quando precisamos redimensionar, isso torna a sua utilização e visualização mais atrativa.

HTML CSS JavaScript Json
Consiste em uma linguagem de marcação utilizada para produção de páginas na web, que permite a criação de documentos que podem ser lidos em praticamente qualquer tipo de computador e transmitidos pela internete. O CSS define como serão exibidos os elementos contidos no código de uma página da internet e sua maior vantagem é efetuar a separação entre o formato e o conteúdo de um documento. É uma linguagem de programação que permite a você criar conteúdo que se atualiza dinamicamente, controlar múltimídias, imagens animadas, e tudo o mais que há de intessante. JSON, em seu significado teórico é "Javascript Object Notation", do qual nada mais é que o formato mais leve conhecido por mim de transferência/intercâmbio de dados, ele é similar ao XML
